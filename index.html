1. We created local git repo
  2. create a remote repo to publish our local repo
  3. configure a remote for our local repo - link the local repo to the remote repo on GitHub
  4. rename our local repo's master branch to "main"
  5. once remote is configured and linked, we add files to staging,
  6. Then we commit the changes
  7. Finally and push files to remote using: "git push -u origin main"
  8. Go back to remote repo on GitHub and hit refresh to see changes
